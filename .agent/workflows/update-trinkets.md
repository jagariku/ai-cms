## 装飾品データの更新手順

装飾品はシーズンごとに更新されるため、以下のスクリプトを使用して最新の日本語データを取得し、記事に反映させます。

### 1. 最新の日本語データを取得する

APIを使用して、特定のDBF ID（または一括取得が機能する場合はタイプ指定）で日本語データを取得します。

```bash
# 取得用スクリプトの実行（例）
ruby fetch_trinkets_ja_direct.rb
```

> [!NOTE]
> `cards.json`（英語版）から `type: 'BATTLEGROUND_TRINKET'` を抽出して最新の DBF ID リストを作成し、それを元に `ja_JP` ロケールで API を叩くのが最も確実です。

### 2. データのフォーマットと分類

取得した JSON データ（`_data/trinkets_ja_sample.json` など）を確認し、「初級（Lesser）」と「上級（Greater）」に分類します。
- `spellSchoolId: 11` -> 初級装飾品
- `spellSchoolId: 12` -> 上級装飾品

### 3. 記事への反映

Markdown 記事の末尾に、以下の形式でテーブルまたはリストとして追記します。

```markdown
## 装飾品一覧

### 初級装飾品
| 名称 | 効果 |
| :--- | :--- |
| [名称] | [効果の説明] |

### 上級装飾品
| 名称 | 効果 |
| :--- | :--- |
| [名称] | [効果の説明] |
```

### 4. 用語のバリデーション

`.github/copilot-instructions.md` に従い、以下の用語が正しく使われているか確認してください。
- 酒場グレード（Tavern Tier）
- 入替（Refresh）
- 聖なる盾（Divine Shield）
- 各種族名（キルボア、マーロック等）
